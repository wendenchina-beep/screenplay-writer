# 共享词典 · 转场词典

3 个 Mode 都会查这个词典。转场（Transition）在剧本里通常靠右显示，全大写。

## 9 种通用转场

| ID | 语义 | 简体中文 | English | 繁體中文 | 日本語 | 한국어 |
|----|------|--------|---------|---------|--------|--------|
| `cut-to` | 硬切 | `切至：` | `CUT TO:` | `切至：` | `カット・トゥ：` | `컷 투:` |
| `dissolve-to` | 叠化 | `叠化至：` | `DISSOLVE TO:` | `疊化至：` | `ディゾルブ：` | `디졸브 투:` |
| `fade-in` | 淡入 | `淡入：` | `FADE IN:` | `淡入：` | `フェードイン：` | `페이드 인:` |

## 使用场景

| 转场类型 | 适用 |
|---------|------|
| **CUT TO** | 默认硬切，最常用。90% 的转场都是它，不需要显式写 |
| **DISSOLVE TO** | 时间跳跃 / 梦境切现实 / 抒情转折 |
| **FADE IN** | 剧本开场（一般是全片第一个词） |
| **FADE OUT** | 剧本结束 / 一幕结束 |
| **FADE TO BLACK** | 强调结尾 / 死亡 / 意识丧失 |
| **MATCH CUT TO** | 图形/动作/主题匹配切换（如太空船切轮子） |
| **JUMP CUT TO** | 时间跳跃（同场景内） |
| **SMASH CUT TO** | 剧烈反差的强切（喜剧/惊悚常用） |
| **WIPE TO** | 老电影感 / 复古风格 |

## 常见规范化前后对照

| 用户原文 | 规范后（中文） | 规范后（英文） |
|---------|-------------|--------------|
| `切换到` | `切至：` | `CUT TO:` |
| `画面切换` | `切至：` | `CUT TO:` |
| `转到下一场` | `切至：` | `CUT TO:` |
| `渐变` / `溶解` | `叠化至：` | `DISSOLVE TO:` |
| `淡入画面` | `淡入：` | `FADE IN:` |
| `画面黑掉` | `淡出至黑。` | `FADE TO BLACK.` |
| `Cut To：` / `Cut to:` | `切至：` | `CUT TO:` |

## 转场元素的排版规则

- 位置：**靠右** align（好莱坞 / 东亚 / 舞台剧 / 广播剧都一样）
- 大小写：**全大写**（除了中文/繁中/日文/韩文，因为这些语言没有大小写概念）
- 前后间距：前 16px、后 0px
- 中英文标点：中文用全角冒号 `：`，英文用半角 `:` + 空格
